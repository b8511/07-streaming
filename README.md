# Week 7 — Streaming

Week 7 of the [DataTalks.Club Data Engineering Zoomcamp](https://github.com/DataTalksClub/data-engineering-zoomcamp/tree/main/07-streaming).

This week covers real-time data streaming using **Redpanda** (a Kafka-compatible broker) and **Apache Flink** via PyFlink. I'm learning how to produce and consume streaming events, and how to apply windowed aggregations — using NYC Green Taxi trip data as the dataset. Everything runs locally with Docker Compose.
